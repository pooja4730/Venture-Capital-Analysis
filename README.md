# Venture-Capital-Investment-Analysis
Exploratory Data Analysis of Venture Capital investments using Python

📌 Executive Summary

This project analyzes global Venture Capital (VC) investment data to uncover funding trends, geographic concentration, and sector-level investment behavior.

Using Python-based data analysis techniques, the project identifies patterns in startup funding across time, countries, and industries to support data-driven investment insights.

🎯 Business Objective

Venture capital funding influences innovation, startup growth, and economic development.
However, funding distribution is uneven across markets and geographies.

This analysis aims to:

Identify long-term funding trends

Determine which countries dominate VC investment

Discover high-performing sectors

Analyze startup funding behavior

Understand funding distribution patterns

📂 Dataset Overview

Source: Kaggle

Total Records: 16,773 startups

Cleaned Records Used: 13,888

Features: 40 variables including funding type, funding rounds, market, country, and funding dates

Note: Dataset is not included due to licensing.
Download from Kaggle and place inside the data/ folder.

📊 Key Insights

Venture capital funding experienced rapid growth in the early 2010s.

The United States dominates global VC investments.

Technology and software-oriented markets receive the highest funding volumes.

Funding distribution is highly skewed — a small number of startups receive disproportionately large investments.

Most startups secure limited funding rounds, indicating selective capital allocation.
